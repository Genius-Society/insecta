# 简介
本“昆虫图鉴数据集”是一个高质量、经标准化处理的专业图像数据集，内含全面涵盖昆虫纲多个目、科、属、种的已脱敏JPG图片，每张图片均配有经昆虫学专家审定的中文名称及拉丁学名双重权威标签；该数据集严格遵循数据合规与隐私保护原则，移除了所有背景及元数据信息，专注于昆虫本体形态特征，旨在为计算机视觉领域的图像分类、目标检测、细粒度识别等模型训练与评估，以及生物多样性研究、数字标本馆建设、智能识别系统开发等多学科交叉应用，提供坚实、可靠且可直接使用的基准数据资源。

昆虫图鉴音频数据集是一套面向智能识别与生态监测的声学样本库，收录中国境内常见昆虫科属的高清野外鸣声录音，涵盖蝉、蟋蟀、螽斯、蜂类等鸣虫的振翅声、求偶声及警戒声等多种行为声学特征，每段音频均配有昆虫图片、中文俗名、拉丁学名等元数据，支持声纹识别、物种鉴定、行为分析、环境声学监测及跨模态联合检索等任务，适用于移动端听虫应用开发、生物多样性声学研究及自然教育场景，助力实现"闻声识虫、声景感知"的智能生态认知体验。

昆虫图鉴视频数据集是一套面向智能识别与生态研究的多模态影像库，收录中国境内常见昆虫科属的高清野外实拍片段，每段视频均配有中文俗名、拉丁学名标注，支持目标检测、行为分析、细粒度分类及跨模态检索等任务，适用于移动端识虫应用开发、生物多样性监测及科普教育场景，助力实现动态化、智能化的昆虫认知体验。
